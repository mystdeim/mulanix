Добавление предложения FROM

Указываем таблицу:

  1.  //SELECT table.* FROM table
  2.  $db->select()
  3.     ->from('table', '*')
  4.     ->query();

Указываем таблицу и столбцы:

  1.  //SELECT table.id, table.text FROM table
  2.  $db->select()
  3.     ->from('table',
  4.            array('id', 'text'))
  5.     ->query();

Указываем таблицу и псевдонимы столбцов:

  1.  //SELECT table.id AS i, table.text AS tx FROM table
  2.  $db->select()
  3.     ->from('table',
  4.            array('id' => 'i', 'text' => 'tx'))
  5.     ->query();

Указываем несколько таблиц + в первой уточняем столбцы:

  1.  //SELECT table.id AS 'i', table.text AS 'tx' FROM table, table2
  2.  $db->select()
  3.     ->from('table')
  4.     ->from('table2')
  5.     ->from('table', array('id' => 'i', 'text' => tx)
  6.     ->query();
